Experiment 7: Data Visualization & Networks

Imitate

I used ChatGPT to brainstorm the idea for this experiment. Eventually I decided to plot the data as a visual bar graph.

Integrate

I was thinking was to convert a .csv file to a 2D array and then plot a dynamic bar graph.The data I want to use for this data visualization is experimental data I collected from my lab in my physics class in community college.

Innovate

I ended up placing the headers for the experimental data above the bars for each corresponding data and put a unique color for each header.

Reflection

The presentation of each experimental data and the changes are very clear. Through the visualization I can very intuitively understand the relationship between the changes in the data. If I had been able to create this visualization of the bar graphs at the time, it would have helped me understand this physics experiment even more.
